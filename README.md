Bebo Hello World

Sample app to get started with Bebo Development

#Get the Files

```
git clone <repo url>
cd hello-world
```

#Setup

```
npm install
```


#Compile (Nessecary before pushing to Github!)

```
npm run build
```

#Release

```
git add .
git commit -m "I made a Bebo mod"
git push origin master
```
